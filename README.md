# Настройка репозитория

0. Создать виртуальное окружения
`python3.11 -m venv venv`
`source venv/bin/activate`

1. Установить зависимости
`pip install --upgrade pip setuptools wheel`
`pip install .`

# Запуск тестов
`pytest -v`
